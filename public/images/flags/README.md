# Flag Images

This directory contains flag images for the language switcher.

Required flag images (20x15 pixels, PNG format):

- `ar.png` - Arabic (Saudi Arabia flag or generic Arabic flag)
- `gb.png` - English (Great Britain flag)
- `fr.png` - French (France flag)
- `ru.png` - Russian (Russia flag)
- `cn.png` - Chinese (China flag)
- `es.png` - Spanish (Spain flag)

You can download these flag images from:

- https://flagicons.lipis.dev/
- https://www.countryflags.com/
- Or use any flag icon library

Make sure the images are:

- 20x15 pixels (or maintain 4:3 aspect ratio)
- PNG format for transparency
- Named exactly as listed above
