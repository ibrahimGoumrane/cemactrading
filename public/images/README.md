# CEMAC Trading - Images Directory

This directory contains all images for the CEMAC Trading application:

## Required Images:

- `logo.png` - Company logo (recommended: 200x60px)
- `hero-bg.jpg` - Hero section background image (recommended: 1920x800px)
- `about-image.jpg` - About section image (recommended: 600x400px)
- `service-icons/` - Directory for service icons
  - `transportation.png` (64x64px)
  - `warehouse.png` (64x64px)
  - `consulting.png` (64x64px)
  - `sourcing.png` (64x64px)

## Optimization Notes:

- Use WebP format for better compression
- Implement responsive images with srcset
- Compress images to reduce load time
- Use lazy loading for below-the-fold images
